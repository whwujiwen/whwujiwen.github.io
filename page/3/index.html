基于Java的游戏外挂

其它

一、外挂分类

内存级外挂———仅仅适用于单机游戏(通常使用C++语言别写，由于java时运行在JVM虚拟机上面的) 数据流级外挂——-专门针对网络游戏，在客户端与服务器之间传输数据时进行拦截(非法的，会被封号) 脚本级外挂———略色外挂（案件精灵、简单游）

二、常用的API

使用的API大多数来自java.awt.Robot这个类

void delay(int ms);//当前程序休眠若干秒
Color getPixelColor(int x,int y);//取得给定屏幕坐标像素位置的颜色值
void keyPress(int keycode);//按下某个按键
void keyRelease(int keycode);//释放某个按键
void mouseMove(int x,int y);//将鼠标移动到指定的屏幕位置，可用于程序的自动延时，测试等等。
void mousePress(int buttons);//指定鼠标按钮按下
void mouseRelease(int buttons);//用于鼠标按钮的抬起
void mouseWheel(int wheelAmt);//用于鼠标按钮的滚轮事件

三、模拟按键的操作

模拟一个人不停的按K键

package com.mntc;
import java.awt.*;
import java.awt.event.KeyEvent;
/*
 @author wujiwen
 @data 2019/9/4 - 18:43   
*/

public class Test {
    public static void main(String[] args) throws Exception {
        //创建一个机器人对象
        Robot robot = new Robot();
        //延时5秒钟
        robot.delay(5000);
        while(true) {
            //指定产生0.2~0.4之间的随机数
            double d1 = 0.2 + Math.random()*(0.4-0.2);
            int d2=(int)(d1*1000);
            System.out.println(d1);

            //指定产生0.2~0.4之间的随机数
            double d3 = 0.2 + Math.random()*(0.4-0.2);
            int d4=(int)(d3*1000);
            System.out.println(d1);
            //释放K键
            robot.keyPress(KeyEvent.VK_K);
            robot.delay(d2);
            //释放K键
            robot.keyRelease(KeyEvent.VK_K);
            robot.delay(d4);
        }
    }
}

四、模拟鼠标的操作

package com.mntc;
/*
 @author wujiwen
 @data 2019/9/4 - 20:03   
*/
import java.awt.*;
import java.awt.event.InputEvent;

public class Test2 {
    public static void main(String[] args) throws Exception{
        //创建一个机器人对象
        Robot robot = new Robot();
        //延时5秒
        robot.delay(5000);
        //将鼠标的指针移动到（900，700）的位置
        robot.mouseMove(1000,1000);
        //点击鼠标
        //BUTTON1_DOWN_MASK:表示鼠标的左键
        //BUTTON2_DOWN_MASK:表示鼠标的中键
        //BUTTON3_DOWN_MASK:表示鼠标的右键
        while(true) {
            robot.mousePress(InputEvent.BUTTON1_DOWN_MASK);
            robot.delay(2000);
            robot.mouseRelease(InputEvent.BUTTON1_DOWN_MASK);
            robot.mousePress(InputEvent.BUTTON1_DOWN_MASK);
            robot.delay(2000);
            robot.mouseRelease(InputEvent.BUTTON1_DOWN_MASK);
        }
    }
}

五.获取指定位置的颜色

获取指定位置的颜色

package com.mntc;/*
 @author wujiwen
 @data 2019/9/4 - 20:31   
*/
import java.awt.*;
public class Test3 {
    public static void main(String[] args) throws Exception{
        //创建一个机器人对象
        Robot robot = new Robot();
        robot.delay(3000);
        //将鼠标移动到指定的位置
        robot.mouseMove(650,650);
        robot.delay(2000);
        //获取指定位置的颜色
        Color color = robot.getPixelColor(650,650);
        System.out.println("红色:"+color.getRed());
        System.out.println("绿色:"+color.getGreen());
        System.out.println("蓝色:"+color.getBlue());
    }
}

二、文件类的CMD命令

c:      切换到C盘，其他盘类似
cd      目录名，如cd test，进入test目录
cd /   返回根目录
cd ..  返回上一层目录
cd.>  文件名，如cd.>js.js 创建一个js.js空文件
md   目录名，如md test，建立一个test目录
mkdir 目录名，如mkdir test，建立一个test目录，功能同上
type  文件名，如type index.html，显示文本文件的内容
tree   以树形结果显示出目录
dir    列出目录中的所有文件

五、批处理

获取连接过的WIFI的密码

@echo off
:main
echo 当前电脑连接过的WIFI名:
netsh wlan show profiles
set /p wifi_name=请输入您要查询的WIFI名称:
netsh wlan show profile name="%wifi_name%" key=clear
echo.
echo 您还想查询其它WIFI吗？如果时，请输入Y，否则输入N退出
set /p input=请输入并选择:
if %input%==Y goto main
if %input%==N goto exit
:exit
exit
pause>nul

修改登陆的密码

@echo off
echo ----------------------- 修改用户登陆密码 --------------------------
set /p user_name=请输入当前用户名(可在C盘用户下进行查看):
set /p pwd=请输入修改后的密码:
net user %user_name% "pwd"
pause>nul
